## 角色/权限

在本博客系统中，通常有以下几种角色和权限：

### 角色权限表

|角色|权限|创建方式|
|--|--|--|
|**超级管理员（Super Admin）**|拥有最高权限，可以管理所有用户、内容和设置。| 手动创建|
|**普通管理员（Admin）**|可以管理编辑、文章、部分设置。| 由超级管理员创建|
|**作者（Author）**|可以撰写和发布自己的博客文章。对文章的操作有：创建、编辑、删除。| 由超级管理员或者普通管理员创建|

这些权限ID遵循了uni-admin的规范：

- 使用英文大写字母和下划线
- 采用"模块_操作_对象"的命名结构
- 确保每个权限ID的唯一性
- 便于在代码中进行权限判断和管理

在uni-admin中配置时，可以按照这些权限ID来设置权限名称和备注，确保权限管理的清晰和规范。
